# Kiosko Vertical - Carousel Manual

Mini sitio HTML estático con Bootstrap 5.3.

## Uso rápido

1. Abre `index.html` en el navegador del kiosko.
2. Reemplaza las imágenes dentro de `assets/img/`.
3. Mantén los mismos nombres o actualiza las rutas en el HTML.

## Imágenes esperadas

El proyecto viene con imagenes en la carpeta assets/img
-Slide1.PNG
hasta el Slide10.PNG


## Características

- Pantalla completa.
- Diseñado para kiosko vertical 9:16.
- Sin autoplay.
- Transición manual con:
  - clic/toque lateral izquierdo o derecho,
  - indicadores inferiores,
  - flechas del teclado,
  - swipe táctil.
- Si el navegador se abre accidentalmente en horizontal, conserva el contenido centrado en relación 9:16.
