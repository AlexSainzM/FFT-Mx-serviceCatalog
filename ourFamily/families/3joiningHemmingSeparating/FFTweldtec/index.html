hemtec Single S

Resumen

Cabezal suspendido por resorte. Permite configuraciones modulares, con geometrías de rodillo seleccionables. Ofrece medición de fuerza para control de calidad.

Características clave

  • Soporte de rodillo y rodillo de hembrado seleccionables del sistema modular.
  • Calidad de proceso optimizada y tolerancia a errores.
  • Las tolerancias de las piezas y los componentes pueden ser influenciadas a través del programa del robot.
  • No se requiere suministro de medios adicionales.
  • Medición de la fuerza de hembrado para control de calidad y del proceso (directa e indirecta).
  • Alta calidad gracias al sistema rígido para el pre-hembrado y sistema suspendido para el hembrado final.
  • Procesamiento de diferentes derivados en la misma línea de producción.
  • Producción de diversas geometrías de hembrado.
  • Sistema modular estandarizado.
  • Adecuado para acero y aluminio.

Geometría de hembrado

Hembrado plano, de cuerda y especial en piezas de acero y aluminio.

Diseño modular

hemtec Single F

Resumen

Cabezal rígido, ideal para puertas, cofres y paneles laterales. Es intercambiable con el modelo S sin cambiar el programa del robot.

Características clave

  • Soporte de rodillo y rodillo de hembrado seleccionables a partir de un sistema modular estandarizado.
  • Calidad de proceso optimizada y alta tolerancia a errores.
  • Las tolerancias de las piezas y la calidad de los componentes pueden ser ajustadas mediante el programa del robot.
  • No requiere suministro de medios adicionales.
  • La base rígida puede intercambiarse fácilmente con el modelo FFThemtec Single S sin necesidad de modificar el programa del robot.
  • Sistema rígido para el pre-hembrado y sistema suspendido para el hembrado final, lo que garantiza una alta calidad en todo el proceso.
  • Posibilidad de procesar diferentes variantes de productos en la misma línea de producción.
  • Medición indirecta de la fuerza de hembrado para un control preciso del proceso.
  • Capacidad para producir diversas geometrías de hembrado.
  • Adecuado tanto para acero como para aluminio.
  • Posiciones ventajosas de los ejes del robot para componentes con formas complejas (como piezas con forma de canal).
